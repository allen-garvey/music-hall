# Music Hall

Web app to explore an artist's albums and music tracks, built with Vue.

## Dependencies

* node >= 22

### Script dependencies

* elixir
* ffmpeg

## Getting Started

* `npm install`
* `npm start`

## Deployment

* `npm run deploy`

## How to convert master wav files

* Add link to master wav files `ln -s /path/to/wavs music`
* `time elixir scripts/convert-masters.exs`
